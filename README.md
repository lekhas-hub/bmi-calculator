# BMI Calculator

## Description
This project calculates Body Mass Index (BMI) based on height and weight entered by the user.

## Features
- Enter weight and height
- Calculate BMI instantly
- Shows BMI category

## Concepts Used
- HTML
- CSS
- JavaScript
- Math operations
- Conditions

## How to Run
Open the project in browser and enter your details.
